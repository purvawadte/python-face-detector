# Real-Time Face Detection using OpenCV

This project detects human faces in real-time using a webcam and OpenCV.

## Features
- Real-time face detection
- Webcam integration
- Rectangle around detected faces
- Face detected label

## Technologies Used
- Python
- OpenCV

## Installation

pip install opencv-python

## Run the Project

python facedetection.py
